Chatbot using Pyhton

This Project is based on Tkinter using python for GUI and AI BOT using JSON file for AI. we proposed an system which will work as an application and give users information about present health. The system finds out what the user actually wants by retrieving the semantic meaning of the query. Then the system will pass on this semantic text as input to the pattern matching algorithm. The patching algorithm make user of knowledge that is already fed into it in order to generate a response.
